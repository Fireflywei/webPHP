2.2.14 CSS 构造颜色、背景与图像

1丶设定颜色

1.红色的几种合法定义 ;

		#f00;
		#ff0000;
		Red;
		Rgb(255,0,0);
		Rgb(100%,0%,0%);

2. 十六进制三元组

		第一个字节 : 红色的值 ;
		第二个字节 : 绿色的值 ;
		第三个字节 : 蓝色的值 ;
		简化写法 :#cccccc 可写成 #ccc, #ff6633 可以写成 #f63

3. 使用 17 种具名颜色

		黑色 (black) 000000
		藏蓝色 (navy) 000080
		绿色 (green) 008000
		海蓝色 (teal) 008080
		银色 (silver) 0c0c0c
		蓝色 (blue) 0000ff
		酸橙色 (lime) 00ff00
		浅绿色 (aqua) 00ffff
		绛紫色 (maroon) 800000
		紫色 (purple) 800080
		橄榄绿 (olive) 808000
		灰色 (gray) 808080
		红色 (red) ff0000
		紫红色 (fuchsia) ff00ff
		黄色 (yellow) ffff00
		橙色 (orange) ffa500
		白色 (white) ffffff
